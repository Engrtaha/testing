## Testing

This repository contains small Python examples and experiments, including practice code using the pandas library (for example, working with `Series` and `DataFrame` objects under the `python/` directory).

### Requirements

- **Python**: 3.13 (or any compatible 3.x version)
- **Virtual environment**: Optional but recommended
- **Packages**:
  - `pandas`
  - Any other dependencies you install for your own experiments

If you are using the existing `.venv` in this repo, activate it instead of creating a new one.

### Setup

1. **Clone or open this repo** on your machine.
2. **(Optional) Create and activate a virtual environment**:

   ```bash
   python -m venv .venv
   source .venv/bin/activate  # On Windows: .venv\Scripts\activate
   ```

3. **Install pandas** (if not already installed in your environment):

   ```bash
   pip install pandas
   ```

### Running the examples

- **From the repo root**, you can run any script under `python/`. For example, to run the `Series` methods example:

  ```bash
  python python/series_structure_pandas/common_methods_used.py
  ```

Adjust the path to match other files under `python/` as needed.

### Notes

- This repository is meant for learning and experimentation. Feel free to add more scripts, tweak examples, or extend the structure as you practice.
